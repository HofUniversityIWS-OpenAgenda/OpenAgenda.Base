Class SelectViewHelper

This ViewHelper generates a

<f:form.select name="paymentOptions" options="{payPal: 'PayPal International Services', visa: 'VISA Card'}" value="visa" /> (Generates a dropdown box like above, except that "VISA Card" is selected.)

<f:form.select name="users" options="{userArray}" optionValueField="id" optionLabelField="firstName" /> (Generates a dropdown box, using ids and first names of the User instances.)

<f:form.select property="salutation" options="{salutations}" prependOptionLabel="- select one -" /> (depending on variable "salutations")

<f:form.select name="paymentOption" options="{payPal: 'PayPal International Services', visa: 'VISA Card'}" translate="{by: 'id'}" /> (Generates a dropdown box and uses the values "payPal" and "visa" to look up translations for those ids in the current package's "Main" XLIFF file.)

<f:form.select name="paymentOption" options="{payPal: 'PayPal International Services', visa: 'VISA Card'}" translate="{by: 'id', prefix: 'shop.paymentOptions.'}" /> (Generates a dropdown box and uses the values "shop.paymentOptions.payPal" and "shop.paymentOptions.visa" to look up translations for those ids in the current package's "Main" XLIFF file.)

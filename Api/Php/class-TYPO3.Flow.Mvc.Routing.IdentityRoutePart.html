Class IdentityRoutePart

Identity Route Part This route part can be used to create and resolve ObjectPathMappings. This handler is used by default, if an objectType is specified for a route part in the routing configuration: - name: 'Some route for xyz entities' uriPattern: '{xyz}' routeParts: 'xyz': objectType: 'Some\Package\Domain\Model\Xyz'

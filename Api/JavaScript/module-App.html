(inner) Application

Main module for OpenAgenda. Its used to inject all needed Modules and to configure the App. See the example below, how the app configures the route-provider.
Source:
  • app.js, line 9
Example
Application.config(['$routeProvider',function($routeProvider) {
         $routeProvider.
            when('/dashboard', {
                templateUrl: '/template/dashboard/index.html',
                controller: 'DashboardCtrl',
                label: 'Dashboard'
            }).
            otherwise({
                redirectTo: '/dashboard'
            });
    }]);

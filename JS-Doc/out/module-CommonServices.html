(inner) service("ModalDialog", "$modal", function($modal))

This service provides a generic modal Dialog. It should be used to show a simple dialog.
Parameters:
Name Type Description
"ModalDialog" string Identifier
"$modal" string Injection
function($modal) function
Author:
  • Thomas Winkler
Source:
  • Features/Common/commonServices.js, line 9
Example
var modalOptions = {
     headerText: 'Erfolg',
            bodyText: 'Das Meetings wurde erfolgreich beendet!'
            };
     var modalDefaults = {
            templateUrl: '/template/modaldialog/success.html'
            };
     ModalDialog.showModal(modalDefaults, modalOptions);

Module: Http

Http module for capturing requests Use the configuration property *proxy* to enable this feature. The HttpInterceptor takes care of failed requests. The HttpManager interacts with the HttpQueue to replay requests. The HttpHeartbeat instance checks the Navigator and Server status. Changes to the online status are issued using the $rootScope. $rootScope.$watch('online', function(newValue, oldValue) { ... });
